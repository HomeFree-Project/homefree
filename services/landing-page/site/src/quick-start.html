Quick start

HomeFree is in technical preview. Today it's for users comfortable installing and configuring software on Linux. The web installer aimed at non-technical users is in progress.

1. Pick the hardware

Any x86_64 box with two NICs (one WAN, one LAN), 8 GB+ RAM, and a decent SSD. AMD or Intel. The HomeFree dev rig is a NUC 9 Pro — less expensive boxes work fine.

2. Install Nix on your workstation

HomeFree is deployed from another machine running Nix. If you're already on NixOS you're set. Otherwise follow the installer on Linux or macOS.

3. Clone the sample config

git clone https://git.homefree.host/homefree/sample-config
cd sample-config

4. Edit the config

Open configuration.nix and set your domain, WAN/LAN interfaces, and which services to enable. The file is heavily commented; most users only touch the top section.

5. Boot the target box from a NixOS USB

Burn the minimal NixOS ISO to a USB stick, plug it into the target machine, and boot from it. Make sure secure boot is disabled and the box has an ethernet cable plugged in (Wi-Fi install isn't supported yet).

6. Find its IP

From the booted-NixOS shell on the target:

ip addr show

7. Run the installer

Back on your workstation, in the sample-config directory:

./install.sh

Enter the target's IP when prompted. The script wipes the disk and lays down the HomeFree image. About fifteen minutes.

8. Sign in

The box defaults to 10.0.0.1 on its LAN side:

ssh adminuser@10.0.0.1

Visit https://your-domain in a browser for the admin UI and the app dashboard.

9. What next

  • Verify DDNS is updating your registrar
  • Open the admin UI and enable the services you want
  • Add users in Zitadel — they'll have SSO across every app immediately
  • Configure backups (Backblaze B2 is the default S3 target; any S3 endpoint works)
Stuck? The FAQ covers the common snags, and #homefree:homefree.host on Matrix is where folks hang out.

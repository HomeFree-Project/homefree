OpenSprinkler UI

This host serves the OpenSprinkler web interface locally from your HomeFree box — no connection to ui.opensprinkler.com, works fully offline.

Point your controller here

  1. On your OpenSprinkler controller, open its script-URL page in a browser:
    http://YOUR-CONTROLLER/su
    Use the controller's LAN address — for example opensprinkler.lan or its IP.
  2. Set the Javascript path to:
    …/js
  3. Save, then reload your controller's web page. It now loads the UI from this box instead of the vendor CDN.
Keep this box online. If it's unreachable, the controller's web UI won't load — but your programs and watering continue to run on the controller itself.
